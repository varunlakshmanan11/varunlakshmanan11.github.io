Projects

Multi-Robot Navigation in Dynamic Environments Using Centralized and Decentralized Monte Carlo Tree Search

Monte Carlo Tree Search (MCTS) for multi-robot navigation in a dynamic environment. The simulation features four TurtleBot3 Waffle robots in a Gazebo world that includes moving obstacles, making the task more challenging. The goal is to analyze the efficiency of Centralized MCTS (where a global planner coordinates all robots) versus Decentralized MCTS (where each robot makes independent decisions). The presence of dynamic obstacles introduces real-world complexity, requiring the robots to continuously adapt their paths.

Performance Analysis of Dueling Double DQN on TurtleBot3 for Navigation in Dynamic Environments

This project evaluates and compares the Dueling Double Deep Q-Network (Dueling Double DQN) against Vanilla Deep Q-Network (DQN) for navigation in a dynamic Gazebo simulation environment. Using TurtleBot3 and ROS2, we train and test both models to determine which performs better in terms of obstacle avoidance and goal-reaching efficiency.

Adaptive Text-to-Command Translation for Robot Navigation

Autonomous Mobile Robot Navigation by integrating Natural Language Processing (NLP) for command-based navigation in a simulated environment. The system fine-tunes a T5-Small model to convert natural language instructions into structured navigation commands, enabling a TurtleBot3 robot to execute multi-step navigation tasks.

A* Algorithm Implementation for a Rigid Robot

Implementation of A* Algorithm for a TurtleBot3 to navigate successfully through the custom made environment in Gazebo

Mobile Pick and Place Manipulator

This Project Involves designing a Mobile Pick and Place Manipulator in Solidworks and Simulating it in Gazebo Environment by using Teleop, Open-loop controllers and Inverse Kinematics.

AI Vision for Robots

Built a computer vision system that enables robots to detect objects and perform scene understanding.
