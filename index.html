Projects

Multi-Robot Navigation in Dynamic Environments Using Centralized and Decentralized Monte Carlo Tree Search

Monte Carlo Tree Search (MCTS) for multi-robot navigation in a dynamic environment. The simulation features four TurtleBot3 Waffle robots in a Gazebo world that includes moving obstacles, making the task more challenging.

Performance Analysis of Dueling Double DQN on TurtleBot3 for Navigation in Dynamic Environments

This project evaluates and compares the Dueling Double Deep Q-Network (Dueling Double DQN) against Vanilla Deep Q-Network (DQN) for navigation in a dynamic Gazebo simulation environment.

Adaptive Text-to-Command Translation for Robot Navigation

Autonomous Mobile Robot Navigation by integrating Natural Language Processing (NLP) for command-based navigation in a simulated environment.

A* Algorithm Implementation for a Rigid Robot

Implementation of A* Algorithm for a TurtleBot3 to navigate successfully through the custom made environment in Gazebo.

Mobile Pick and Place Manipulator

This project involves designing a mobile pick-and-place manipulator in SolidWorks and simulating it in Gazebo using Teleop, Open-loop controllers, and Inverse Kinematics.

Gesture-Based Control in Assistive Technology

Gesture-based control for assistive technology using Google's Mediapipe Library to process hand gestures from a webcam. The recognized gestures are mapped to control DexHand in ROS2 simulation and a TurtleBot for intuitive robotic interactions.

Performance Analysis of Variants RRT Path Planning Algorithm

This Project involves comparison of performance of different variants of RRT Path Planning algorithms in 2D and implementing them to a TurtleBot3 in gazebo simulation environment by using a pid controller to check their performance on 3D simulation.

Perception Based Robot Navigation

This project involves developing a perception system for a TurtleBot3 robot to navigate in a dynamic environment. The perception system uses OpenCV to detect and track objects in the environment and provides the robot with the necessary information to navigate safely.
